# Tokoku - Sistem Kasir & Gudang

## Deskripsi
Tokoku merupakan aplikasi Sistem Informasi Kasir & Gudang yang dibangun menggunakan Laravel sebagai Backend API dan HTML, CSS, JavaScript sebagai Frontend.

## Teknologi
- Laravel
- MySQL
- HTML, CSS, JavaScript

## Struktur Project

/
├── tokoku-backend/    # Backend Laravel API
└── index.html         # Frontend

## Cara Menjalankan

1. Pastikan database telah di-import.
2. Jalankan backend Laravel:

   php artisan serve

3. Buka `index.html`.
4. Pastikan backend Laravel tetap berjalan agar frontend dapat mengakses API.
